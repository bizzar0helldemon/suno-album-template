## Workflow Stages

Each track moves through these stages:

```
[Concept] → [Lyrics] → [Design Doc] → [Suno Ready] → [Generated] → [Final Mix]
    1           2           3              4             5             6
```

### Stage Definitions

1. **Concept** — Title exists, themes identified, not yet written
2. **Lyrics** — Full lyrics written, structure determined
3. **Design Doc** — `docs/Tracks/[Track]_Design.md` created with all 11 sections
4. **Suno Ready** — `docs/Tracks/[Track]_Suno_Ready.md` with meta tags, ready for generation
5. **Generated** — Successful Suno generation completed, logged in GENERATION_LOG.md
6. **Final Mix** — Track approved for album, any post-production complete
