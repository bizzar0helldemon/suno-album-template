```
verse-chorus structure, conventional pop structure, radio-friendly arrangement
```

**HYPNOTIC/LOOP-BASED:**
```
repetitive structure, hypnotic loops, minimal variation, trance-like progression
```

**EVOLVING/BUILDING:**
```

```

```

```
